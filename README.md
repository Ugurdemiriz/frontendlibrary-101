## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- Basic understanding of how to use a computer

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ugurdemiriz/frontendlibrary-101.git
   cd frontendlibrary-101
   ```

2. **Open in your browser**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     
     # Using VS Code Live Server extension
     # Right-click index.html > Open with Live Server
     ```

3. **Start Learning**
   - Navigate through the sections using the menu
   - Explore examples in each category folder
   - Practice by modifying the code

## ğŸ“ Learning Path

### 1. HTML Fundamentals
Start with HTML to understand the structure of web pages.

**Topics Covered:**
- HTML document structure
- Semantic HTML5 elements
- Headings, paragraphs, and text formatting
- Lists (ordered, unordered, description)
- Images and media
- Links and navigation
- Forms and input elements
- Tables

**Location:** `01-html-kategorisi/`

### 2. CSS Styling
Learn how to style and layout your web pages beautifully.

**Topics Covered:**
- CSS selectors and specificity
- Box model (margin, padding, border)
- Flexbox and Grid layouts
- Responsive design with media queries
- CSS variables and custom properties
- Animations and transitions
- Best practices and methodologies

**Location:** `02-css-kategorisi/`

### 3. Bootstrap Framework
Build responsive websites quickly using Bootstrap.

**Topics Covered:**
- Bootstrap grid system
- Responsive utilities
- Components (cards, buttons, forms)
- Navigation and navbar
- Modals and dropdowns
- Bootstrap icons and Font Awesome
- Customization and theming

**Location:** `03-bootstrap-kategorisi/`

### 4. JavaScript Basics
Add interactivity and dynamic behavior to your websites.

**Topics Covered:**
- Variables and data types
- Functions and arrow functions
- DOM manipulation
- Event handling
- Arrays and objects
- Async/await and promises
- ES6+ features
- Error handling

**Location:** `04-javascript-kategorisi/`
